# NFSKit
This is small Swift library for iOS which wraps libnfs and allows to amount NFS share and do file operation.

Code heavily borrowed from [AMSM2](https://github.com/amosavian/AMSMB2)

Status
====
This project is still under developing.

Via CocoaPod
